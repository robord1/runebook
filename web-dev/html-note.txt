it is recommended to always pair a target=_blank with a rel=noopener noreferrer.
Relative links only contain filepath to the other page relative to the page where the link is created. 
Absolute links contain domain name and path
If relative link breaks try adding ./ at the start to specify relative to the current directory
To get to the parent directory use ../
